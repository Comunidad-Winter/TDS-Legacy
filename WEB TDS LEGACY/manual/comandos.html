COMANDOS

Los comandos se introducen en la parte de chat y corresponden a códigos útiles para complementar el juego, estos comandos sirven tanto para comerciar, como para mandar mensajes privados, etc. A continuación los comandos referentes a la parte del sistema. Dentro de las secciones que correspondan se explicarán los comandos netamente del juego:

/Ayudas
Te da una ayuda mínima (la forma de mandar mensajes para que los lea un sólo usuario, o escribir en letra roja, etc.). Sirve por si se olvidan de cómo se hace, pero no brinda ninguna información que no esté en este manual.

/Desc
Con este comando puedes cambiar la descripción de tu personaje. /Est
Esto te muestra tu estado: te indica la cantidad de vida, mana y energía que tienes, además de indicarte en que mapa y en que coordenadas te encuentras. Por otra parte, te indica tu mayor y menor golpe y, si tienes un arma equipada, te mostrara en paréntesis sus atributos, Lo mismo sucede con tu defensa.

/GM
Si tienes preguntas que a las que el manual no tenga respuestas, puedes hacerlas usando el comando

/GM seguido de tu pregunta.
La contestación a tu duda se contestará al mail al que esté registrado tu personaje.

/Denunciar
Envía un mensaje por consola a los gms, es instantáneo, y se usa para casos como robos de NPCs, piquetes o personas usando cheats. (No abusar de este comando, porque el mal uso está penado)

\Nick+del+destinatario
Envía mensajes privados a un user en especial.
¿Cómo lo uso?

/Online
Te indica la cantidad de personas en línea con sus respectivos nombres.

/Ping
Te dice el lag.

/Penas
Se utiliza para saber las penas que tiene el personaje.

/Passwd
Con este comando puedes cambiar tu contraseña. Al teclearlo aparecerá un pequeño formulario en e que deberás ingresar tu contraseña actual y la nueva.

/Seg
Este comando activa o desactiva el seguro, cuando el seguro esta activado no le puedes pegar a ningún ciudadano o su mascota aunque el modo combate este activado.

/Salir
Con este comando sales de tu personaje y regresas a la pantalla principal.

/Fundarclan
Sirve para fundar un clan.

/Votar NICK
Sirve para votar el líder de tu clan.

/Onlineclan
Sirve para saber qué miembros de tu clan se encuentran conectados actualmente.

/Cmsg
Sirve para comunicarte con los miembros de tu clan, si es que perteneces a uno.

/Psmg
Sirve para comunicarte con los miembros de tu party, si es que perteneces a una.

/Depositar X
La función de este comando es la de depositar dinero en un banco. Debes reemplazar la X por la cantidad de oro a depositar.

/Depositartodo
Deposita en el banco todo el oro que tienes.

/Retirar X
Sirve para extraer dinero de tu bóveda. Debes reemplazar la X por la cantidad de oro a retirar.

/Retirartodo
Sirve para retirar de la bóveda todo el oro que tienes.

/Balance
Sirve para ver cuánto dinero tienes en el banco.

/Boveda
Accedes al sistema de bóveda, para depositar y extraer objetos.

/Comerciar
Sirve para comerciar con un vendedor o un usuario, en el caso de vendedores tambien se puede hacer doble click sobre ellos.

/Resucitar
El sacerdote de cada ciudad puede revivirte, también puedes hacer doble click sobre él.

/Curar
El sacerdote de cada ciudad puede curarte si estás herido, tambien puedes hacer doble click sobre él.

/Descansar
Sirve para recuperar energía y salud junto a una fogata.

/Entrenar
Muestra la lista de criaturas que el NPC entrenador puede invocar.

/Enlistar
Sirve para enlistarse en la Armada Real o la del Caos.

/Retirar
Sirve para desenlistarse de la Armada Real o la del Caos.

/Recompensa
Si cumples tu objetivo, recibirás una recompensa del Rey/Demonio.

/Meditar
Las clases con maná recuperan la misma con este comando. También puedes usar las teclas F6 o FIN.

/Quieto
Sirve para que tu mascota se quede en el lugar.

/Abandonar
Sirve para abandonar a tu mascota en el lugar que se encuentre

/Cheque Código
Cuando vendes un personaje por MercadoAO ya no tienes acceso a él. Recibirás un mail con el nombre del comprador y un código con el cual podrás retirar el dinero en cualquier banco. Cuando estes con el banquero, pondras el comando, siendo "código" el código que llegó a tu mail.

/Activar
Sirve para activar el chat Global, con el cual podrás hablar con cualquier jugador del servidor.
Cada vez que quieras hablar por el chat global deberás colocar un punto antes de empezar a escribir.
